# Cohort 3 Template

A simple and clean static website template created for Cohort 3. 

## 🚀 Live Demo
[View the live project here](https://cohort3-templete.vercel.app/) 

## 🛠️ Tech Stack
* **HTML5:** For the page structure and content.
* **CSS3:** For styling and layout.
* **Typography:** Custom font (Montserrat).

## 📂 Project Structure
* `index.html` - The main HTML document.
* `style.css` - The stylesheet containing all the design rules.
* `Montserrat-Regular.ttf` - The custom font file used in the project.
